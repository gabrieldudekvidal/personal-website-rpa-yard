Date

It's possible to add symbols ( -, / ...) for data format
DateTime.Now.AddDays(0).ToString("dd-MM-yyyy") -- Current date

DateTime.Now.AddDays(-1).ToString("dd/MM/yyyy") -- Yesterday's date

DateTime.Now.AddDays(+1).ToString("dd/MM/yyyy") -- Tomorrow's date

DateTime.Now.ToString("MM/yyyy") -- Month and Year

DateTime.Now.ToString("01/MM/yyyy") -- Fist day of current month

(New DateTime(Today.AddMonths(1).Year,Today.AddMonths(1).Month,1)).AddDays(-1).ToString("dd/MM/yyyy") -- Last day of current month

DateTime.Now.AddMonths(-1).ToString("01/MM/yyyy") -- 'First day of previous month

New DateTime(now.Year,now.Month,1).AddDays(-1).ToString("dd/MM/yyyy") -- Last day of previous month

DateTime.Now.AddDays(0).ToString("dddd") -- Get day name

DateTime.Now.AddMonths(0).ToString("MMMM") -- Get month name
It's possible to add symbols ( -, / ...) for data format

Time

DateTime.Now.AddHours(0).ToString("hh:mm:ss") -- Current hours, minutes, and seconds.
It's possible to get less and more hours by adding: - or + inside of ().

Killing Process

A good way to close applications is using cmd and typing the following command:
taskkill /f /im process.exe
e.g.
taskkill /f /im chrome.exe
